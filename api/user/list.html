获取用户列表 #

  • 请求方式: GET
  • 请求地址: /api/users

参数名类型是否必传说明
keywordStringNO搜索关键词，搜索字段：users.name

json
{
    "err_code": 0,
    "err_msg": "success",
    "data": [

            // API 参考 > 接口数据模型 > 用户信息,

  • 查询用户表的所有信息
